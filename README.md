## Technologies
- **Android**
- **Java**
- **```RecyclerView``` with ```StaggeredLayoutManager```**
- **Room Database**
## Features
- ***Add***, ***Update*** and ***Delete*** a note.
- Add an ***Image*** to the note.
- Add a ***Link*** to the note.
- Add different ***Colors*** the note.
- ***Search*** through All the notes.
